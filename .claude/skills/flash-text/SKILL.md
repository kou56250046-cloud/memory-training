# 瞬読フラッシュテキスト生成スキル

## 概要

このスキルは `training-prototype.html` のフラッシュトレーニングで使用する
練習テキストを生成します。`texts/[slug].txt` に保存し、
`texts/index.json` にメタ情報（タイトル・設問）を追記します。

---

## 生成仕様

| 項目 | 内容 |
|------|------|
| 文字数 | 3000〜5000字（本文のみ） |
| 言語 | 日本語 |
| **改行** | **30文字ごとにハード改行（\n）を入れる** |
| 保存先 | `texts/[slug].txt` |
| 索引更新 | `texts/index.json` に追記 |

---

## 実行手順（8ステップ）

### Step 1：テーマ確認

引数があればそれを使用。なければ以下から選ぶか提案する：
- 脳・感覚・知覚
- 習慣・行動・変化
- 学習・記憶・集中
- 自然・環境・身体
- 人間関係・感情・共感
- 時間・朝・夜・季節

### Step 2：感情設計

3段階の感情の流れを設計する：
1. **冒頭**：「あ、これ自分に関係ある」という共感
2. **中盤**：「へえ、そういうことか」という納得・軽い驚き
3. **末尾**：「試してみようかな」という前向きな気づき

### Step 3：構成を決める

```
導入（150字）：具体的な情景・問い・驚きの事実から入る
本題（2500〜4300字）：3〜5つの柱で展開
締め（200字）：問いかけか短い気づきで終わる
```

### Step 4：文体ルールを守る

- **語り口**：「研究者が友人に話すように」
- 1文が30〜60字程度
- 専門用語は初出時に「平易語（専門用語）」の順で書く
- 抽象的な話には必ず具体例かたとえを添える
- 断定調（「〜だ」「〜である」）だが押しつけない
- 箇条書きより文章の流れを優先する

### Step 5：「斜め上の視点」を1箇所入れる

以下いずれかのパターン：

| パターン | 例 |
|---------|----|
| 時間軸をずらす | 「現代の悩みが数万年前と同じ理由」 |
| 因果を逆転させる | 「集中するから楽しいのではなく、楽しいから集中できる」 |
| 悪役を主人公にする | 「忘れることは脳の優秀な機能だった」 |
| 別分野の知見で見る | 「この問題を生態学で解くと〜」 |

### Step 6：たとえを1つ以上入れる

良いたとえの条件：
1. 読者が「見たことある」ものを使う
2. たとえの後に「つまり〜だ」で本題に戻す
3. 2〜3行以内に収める

### Step 7：執筆と改行フォーマット

**重要：30文字ごとに改行を入れる。**

方法：
1. まず文章を自然に書く
2. 書きながら30字で改行を入れる
3. 文の途中でも30字になったら改行する
4. 句読点の直後が30字付近なら、そこで改行する（読みやすさ優先）

例：
```
視覚、聴覚、嗅覚、味覚、触覚。人間は
この五つの感覚器官を通じて外界の情報
を受け取り、世界を認識している。しか
し現代社会において、私たちの五感はか
つてないほど刺激過多の状態に置かれて
いる。
```

### Step 8：保存と index.json 更新

1. `texts/[slug].txt` に本文を保存
2. `texts/index.json` を読み込む
3. 以下の形式でエントリを追記して保存：

```json
{
  "id": "slug",
  "title": "タイトル（20字以内）",
  "file": "texts/slug.txt",
  "category": "脳・感覚 | 習慣 | 学習 | 自然 | 人間関係",
  "questions": [
    {
      "q": "設問文（文章の本質・主旨を問う）",
      "choices": ["選択肢A", "選択肢B", "選択肢C", "選択肢D"],
      "answer": 0
    }
    // 計4問
  ]
}
```

**4択設問のルール**：
- 文章を読んだ人なら答えられる内容（本文に根拠がある）
- 「細かい数字の暗記」ではなく「主旨・意図・対比」を問う
- 誤答の選択肢は「それっぽいが違う」ものを選ぶ
- 正解はランダムに 0〜3 の位置に置く

---

## ファイル命名規則

- slug は英小文字・ハイフン区切り（例：`gokan`, `kioku-gijutsu`）
- タイトルは日本語20字以内

---

## 使用例

```
/flash-text 朝の習慣と脳の働き
/flash-text
/flash-text 孤独と人間関係の心理学
```
