# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

**瞬読・記憶法トレーニングサイト** — 瞬読（速読）と記憶法を「学べて」「鍛えられる」一体型 Web トレーニングサイト。

- **形式**: 単一 HTML ファイル（HTML + CSS + JavaScript を内包）
- **外部依存**: Chart.js（CDN のみ）
- **データ保存**: localStorage（端末内、サーバー不要）
- **ホスティング**: GitHub Pages（静的）
- **ユーザー**: 完全個人運用・単独利用・完全無料

## 開発の進め方

ビルドツール・パッケージマネージャ不要。HTML ファイルを直接編集してブラウザで確認する。

```bash
# GitHub テキスト連携を含む動作確認（CORS回避のため簡易サーバー推奨）
python -m http.server 8000
# または
npx serve .
```

ファイルを直接開く場合（`file://`）は、GitHub fetch が CORS でブロックされる点に注意。
fetch 不要な機能（フラッシュ・単語・数字記憶など）はファイル直開きでも動作する。

## サイト構成（UI）

3タブ構成。各セクションは **カード一覧 → 詳細画面** のナビゲーション。

| タブ | コンテンツ |
|-----|-----------|
| ホーム | サイト目的・使い方・各セクションへの導線（CTA） |
| 瞬読 | 解説 + 6種トレーニング + 実力チェック + 記録 |
| 記憶法 | 解説 + 4種トレーニング（+ 3種は解説のみ） + 実力チェック + 記録 |

詳細画面の構造:
- カードクリック → スライドインで開く
- 上部: 意義・やり方の解説（文章形式、箇条書き羅列にしない）
- 下部: トレーニング本体
- 「← 一覧へ」でカード一覧に戻る

## トレーニング仕様

### 瞬読（6種）

| # | 名称 | 素材 | 備考 |
|---|------|------|------|
| 1 | フラッシュ | 読込テキスト（GitHub選択 / 貼付 / .txt読込） | |
| 2 | 単語 | 内蔵単語リスト（4〜10字、約270語） | 横書き/縦書き切替、文字順シャッフル |
| 3 | 円形 | 内蔵単語リスト | 文字順シャッフル |
| 4 | ランダム配置 | 内蔵単語リスト | |
| 5 | 観察力 | 数字グリッド（2×2〜4×4）自動生成 | |
| 6 | イメージ力 | 内蔵語 | |

瞬読の共通設定: フォント倍率（0.8〜2.5）・右脳沈黙モード（黒画面・秒数設定）・サンプルテキスト選択。

### 記憶法（4種トレーニング + 3種解説のみ）

| # | 名称 | 備考 |
|---|------|------|
| 解説 | 場所法・記憶の宮殿 | トレーニングなし |
| 解説 | ジャーニー法 | トレーニングなし |
| 解説 | ストーリー法・連想 | トレーニングなし |
| 1 | 数字記憶 | 桁数・表示秒数・区切り設定、位置一致採点 |
| 2 | 単語記憶 | 内蔵リストから出題、想起入力（順不同採点） |
| 3 | カード配置記憶 | トランプ（♠♥♦♣ A〜K）全表示→伏せ→ペア当て。ペア数・記憶秒数設定可 |
| 4 | ドミニクシステム | 変換表 + 個人テーブル（00〜99、空から入力）+ ドリル |

記憶法の実力チェックは **種目選択式**（数字記憶20桁30秒 / 単語記憶15語40秒 / カード配置8ペア）。

## localStorage スキーマ

| キー | 内容 |
|-----|------|
| `shun_history` | 瞬読履歴（日時・種類・速度/結果）。実力テスト結果も含む |
| `mem_history` | 記憶法履歴（日時・手法名・正答率・スコア）。実力テストは手法名 `実力テスト(数字)` のように区別 |
| `dominic_table` | ドミニク個人テーブル（00〜99 の人物/行動/物） |

旧キー（`trainingHistory` / `digit_history`）からの移行は不要。

## GitHub サンプルテキスト連携

`texts/index.json` でメタ情報を管理。起動時に fetch してドロップダウンを生成。

```json
[
  { "id": "gokan", "title": "五感を研ぎ澄ます効果", "file": "texts/gokan.txt", "category": "エッセイ" }
]
```

- `raw.githubusercontent.com` から GET 取得（CORS 許可済み）
- fetch 失敗時は内蔵サンプルをフォールバックとして使用
- テキスト追加時は `index.json` と `.txt` を同時にコミット

## デザイン規約

- **ライトテーマのみ**（ダーク不要）
- 瞬読 = 深緑 / 記憶法 = テラコッタ でアクセントカラーを分ける
- 可読性重視（明朝見出し + ゴシック本文）
- アニメーション: カードの時間差フェードイン・ホバー浮き上がり・詳細スライドイン・カードのポップ/フリップ
