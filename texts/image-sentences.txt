# イメージ力トレーニング用短文リスト（20〜30字）
# 1行1文、#始まりはコメント行
# 場面が頭に浮かぶ具体的な情景描写文

# 日常・人物
毎朝五時に起きて近所の川辺を散歩する
彼女は突然のプレゼントにとても喜んでくれた
老いた犬が縁側で気持ちよさそうに眠っている
少年は泥だらけの手で野球ボールを握りしめた
白髪の祖母がゆっくりと縁側でお茶を飲んでいる
青年は夜の図書館でひとり本を読み続けていた
女の子が赤い長靴で水たまりをぴちゃぴちゃ踏む
父親が子供の肩に手を置き優しく微笑んでいる
母は台所で鼻歌を歌いながら夕飯を作っていた
老人が公園のベンチに座り鳩に餌をやっている
子供たちが校庭で笑いながら追いかけっこをする
青いワンピースの女性が駅のホームで本を読む
少女は川の石を拾って丁寧に並べて飾っていた
男の子が父の大きな背中にしがみついて笑っている
祖父は縁側に座り夕焼けをじっと眺めていた

# 自然・風景
夕暮れの浜辺に老夫婦が並んで座っていた
窓辺の猫が降り始めた雨をじっと眺めていた
春の公園で子どもたちが元気よく走り回っていた
白い犬が尻尾を振りながら子供に駆け寄った
川沿いの桜並木に花びらが静かに舞い落ちていく
夜の海岸に打ち寄せる波の音だけが響いていた
霧の中に古い灯台がぼんやりと浮かび上がっていた
朝露に濡れた草むらを素足で歩く少年がいた
夕焼けに染まる空の下で子供たちが遊んでいた
雪山の頂上から見える白い雲の海が広がっていた
田んぼのあぜ道をのんびり歩くおじさんがいた
竹やぶの間から細い光の筋が差し込んでいた
湖面に映る満月が風でゆっくりと揺れていた
朝靄の中を自転車で走る学生の後ろ姿があった
砂浜に残された子供の足跡が波に消えていった

# 動物・生き物
赤いセーターの女性が傘を差して雨の中を歩く
白い馬が草原を風のように走り抜けていった
老木の上でフクロウが月を見上げて鳴いていた
川で釣りをする少年の横にアヒルが並んで立つ
野良猫が段ボール箱の中で子猫を育てていた
黒いラブラドールが砂浜でボールを追いかけた
竹林の中を鹿が静かに横切って消えていった
橋の下で子猫がひとり雨宿りをしていた
庭の池で錦鯉がゆっくりと泳ぎ回っていた
蝶々が花から花へとひらひら移っていった

# 季節・行事
古い木の椅子に青年が一冊の本を持って座っている
お祭りの夜店で金魚すくいに夢中になる兄弟がいた
雪の中で子供たちが雪だるまを作って笑っていた
真夏の砂浜でスイカを頬張る家族の笑顔があった
紅葉した山道を老夫婦が手をつないで歩いていた
節分の豆まきで弟が鬼の面をつけて泣いている
花火大会の夜空を仰ぎ見る浴衣の女の子がいた
師走の商店街を急ぎ足で通り過ぎる人々がいた
お正月に家族が囲炉裏の周りで笑い合っていた
春の野原でシロツメクサの輪を作る少女がいた

# 食べ物・料理
熱々のラーメンをふーふーしながら食べる少年
おばあちゃんが台所で餅をついて笑っている
縁側で食べるスイカの種を遠くに飛ばした
母の握ったおにぎりを山の頂上で頬張った
焼きたてのパンの香りが路地に漂っている
夕食のシチューが煮える音と香りが広がる
かき氷屋の前に子供たちの行列が並んでいた
おじいちゃんが孫に手作りのみたらし団子を渡す

# 場所・空間
古い木造の校舎に夕日が斜めに差し込んでいた
石畳の路地を黒猫が音もなく歩いていった
夜の駅のホームにひとりの旅人が立っていた
廃線の線路の上を子供たちが歩いて遊んでいた
図書館の奥の席で老人が分厚い本を読んでいた
山小屋のランプの光の中で家族が食卓を囲んでいた
雨の夜のカフェで二人がコーヒーを飲んでいた
古い神社の石段を女の子がひとり登っていた
病院の廊下でお父さんが赤ちゃんを抱っこして歩く
夕暮れの公園で少年がひとりブランコを漕いでいた

# 感情・心情
失恋した夜に星空を見上げてため息をついた
初めて自転車に乗れた日の誇らしい気持ち
試験に合格した瞬間に涙がこぼれてしまった
遠くに引っ越す友達に手を振って別れを告げた
久しぶりに会った祖母が私の手をそっと握った
