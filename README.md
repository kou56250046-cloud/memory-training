# 瞬読・記憶法トレーニングサイト

瞬読（速読）と記憶法を「学べて」「鍛えられる」一体型 Web トレーニングサイト。

## 特徴

- **解説 + トレーニング一体型**: 各手法の「なぜ効くのか」「どう使うか」を読んでから練習できる
- **瞬読 6種 / 記憶法 4種**: フラッシュ・単語・円形・ランダム配置・観察力・イメージ力 / 数字記憶・単語記憶・カード配置記憶・ドミニクシステム
- **実力チェックテスト**: 条件固定の腕試しで現在地を把握、称号付与
- **履歴・グラフ**: 種別・日別の推移グラフ、CSV 出力
- **インストール不要**: 単一 HTML ファイルをブラウザで開くだけ

## 使い方

```
training-site.html をブラウザで開く
```

GitHub テキスト連携（フラッシュのサンプルテキスト選択）を使う場合は、ローカルサーバー経由で開く:

```bash
python -m http.server 8000
# → http://localhost:8000/training-site.html
```

## サンプルテキストの追加

`texts/` ディレクトリに `.txt` ファイルを置き、`texts/index.json` に追記する:

```json
[
  { "id": "gokan", "title": "五感を研ぎ澄ます効果", "file": "texts/gokan.txt", "category": "エッセイ" }
]
```

## データ保存

履歴・設定・ドミニク個人テーブルはすべてブラウザの localStorage に保存。
サーバー通信なし。CSV エクスポートでバックアップ可能。

## ホスティング

GitHub Pages などの静的ホスティングに対応。
`training-site.html` と `texts/` フォルダをそのままデプロイするだけ。
